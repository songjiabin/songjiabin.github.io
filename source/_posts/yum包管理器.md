> Yunm是一个shell前端软件包管理器，基于RPM包管理，能够从指定的服务器自动下载RPM包并且安装，可以自动处理依赖性关系，并且一次安装所有依赖软件包。




##### 查看yum 服务器所有的包
```
yum list 
```
##### 安装火狐
```Java
1、先查看火狐在yum服务器是否有
2、yum install firefox 
```
